# Private Rooms

## Explanation
In Private Rooms players either play a normal Ranked Seed or they can choose a Set Seed. It is also possible to disable Seedtypes so players can practice the Seedtypes they want. There are 20+ Categories to choose from aside from the standard Any% Category and players can chooe which Rules to turn ON/OFF (QOL rules, experimental features and rules that make the game harder/easier(fairer). 

## Special Tiers
Everyone can make a Private Room in the most basic form, but certain features require a [Special Tier](https://store.mcsrranked.com/category/tiers). Here is specified what the different tiers can do in a Private Room.

### No Pickaxe
- Max. 32 players
- Toggle Rules ON/OFF
- Run Command on World Join
- Choose Category
- Choose a Set Seed

### Stone Pickaxe
- Same as No Pickaxe
- Solo Private Room
- Choose Seedtype

### Iron Pickaxe
- Same as Stone Pickaxe
- Max. 64 players
- Choose a Custom Room Name
- Play against Ranked Bot

### Diamond Pickaxe
- Same as Iron Pickaxe
- Request to Join Private Room option
